Newo AI Receptionist
====================

Free connector between Odoo and the Newo AI Receptionist SaaS platform.

Setup
-----

1. Sign up at https://newo.ai and obtain an API key.
2. In Odoo, go to **Settings → Newo AI Receptionist** and paste the key.
3. Click **Connect**. The module will register a webhook with Newo and verify the link.

Features
--------

* Inbound call answering by the AI agent.
* Automatic ``calendar.event`` creation for bookings.
* Automatic ``crm.lead`` creation with transcript and recording URL.
* ``res.partner`` deduplication by phone and email.

Subscription required
---------------------

Module install and configuration are free. To run any conversation, an active
Newo subscription is required. See https://newo.ai/pricing.
